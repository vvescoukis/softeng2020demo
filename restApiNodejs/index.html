Supported Endpoints
1) http://restdemo.softlab.ntua.gr/data/
Description: Returns all the charging stations stored in mySQL server
Parameters : None
2) http://restdemo.softlab.ntua.gr/data/municipalities
Description: Returns all the distinct municipalities of all charging stations stored in mySQL server
Parameters : None
3) http://restdemo.softlab.ntua.gr/data/names
Description: Returns all names of all charging stations stored in mySQL server
Parameters : None
4) http://restdemo.softlab.ntua.gr/data/name/Stovolou - Kantaras Junction
Description: Returns all the data about a specific charging station
Parameters : name of Charging Station --> http://restdemo.softlab.ntua.gr/data/name/ + the name of the station
5) http://restdemo.softlab.ntua.gr/data/municipality/Strovolos
Description: Returns all charging stations of a specific municipality
Parameters : municipality name --> http://restdemo.softlab.ntua.gr/data/municipality/ + the name of the municipality
You can also limit the size of the response by adding ?limit=x where x is an integer example http://restdemo.softlab.ntua.gr/data/municipality/Strovolos?limit=1
